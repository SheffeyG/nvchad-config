# NvChad custom config

## NvChad setup

```
git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1
```

## setup config
```
rm -r ~/.config/nvim/lua/custom/* 
git clone git@github.com:SheffeyG/nvchad-config.git ~/.config/nvim/lua/custom
```
